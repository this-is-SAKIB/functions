Chapter 01

Monotonicity & Intervals

A function is increasing where its graph rises left-to-right, and decreasing where it falls. The derivative tells you the slope at every point — so the sign of \(f'(x)\) reveals everything.

The Core Rule
\(f'(x) > 0\) on \((a,b)\) \(\Rightarrow f\) is increasing on \([a,b]\)
\(f'(x) < 0\) on \((a,b)\) \(\Rightarrow f\) is decreasing on \([a,b]\)
\(f'(x) = 0\) at \(x=c\) \(\Rightarrow\) potential turning point (or flat section)

Note: we state the intervals using closed brackets \([a,b]\) in the final answer, even though the sign test uses open intervals. (The endpoint values themselves don't affect which way the function runs.)

The Sign Chart Method — Step by Step

  1. Find \(f'(x)\).
  2. Set \(f'(x) = 0\) and also find where \(f'(x)\) is undefined. These are your critical numbers (and also any points not in the domain of \(f\)).
  3. Plot all those values on a number line. They divide the line into intervals.
  4. Pick one test value inside each interval and evaluate \(f'(\text{test})\).
  5. Record + or − for each interval. That tells you increasing or decreasing.
Example 1 \(f(x) = x^3 - 3x + 2\)

\(f'(x) = 3x^2 - 3 = 3(x-1)(x+1)\)

Critical numbers: \(f'(x)=0 \Rightarrow x = -1,\; x = 1\). No points where \(f'\) is undefined.

Interval \((-\infty,-1)\) \(x=-1\) \((-1,\,1)\) \(x=1\) \((1,+\infty)\)
Test value \(x=-2\) — \(x=0\) — \(x=2\)
\(f'(x)\) + (9) 0 − (−3) 0 + (9)
\(f\) is… ↗ Increasing — ↘ Decreasing — ↗ Increasing

Conclusion: \(f\) is increasing on \((-\infty,-1]\) and \([1,+\infty)\); decreasing on \([-1,1]\).

⚡
Shortcut for Polynomials

After you factor \(f'(x)\), use the zero-cross rule: each simple root of \(f'\) flips the sign. Double roots do NOT flip the sign. So you only need to evaluate \(f'\) once if you know the roots' multiplicities.

🔍
Where f ' is Undefined

Always include points where \(f'\) doesn't exist (cusps, corners, vertical tangents) in your sign chart, even if \(f\) is still defined there. The function can change direction there too.

⚠️
The Constant Trap

If \(f'(x)=0\) on an entire interval (not just a point), \(f\) is constant there — neither increasing nor decreasing. Don't call it "increasing" just because the derivative is non-negative.

📝
Closed vs Open Intervals

Write the final monotonicity intervals with closed brackets at the endpoints in most textbooks. Some courses use open; check yours. The logic is the same either way.

Chapter 02

Local (Relative) Extrema

A local maximum is a peak — \(f(c)\) is the highest value in some neighbourhood. A local minimum is a valley. You find them at critical numbers where \(f'\) changes sign.

First Derivative Test (FDT)

At a critical number \(c\) where \(f\) is continuous:

Sign of \(f'\) changes at \(c\)Conclusion
+ → − (positive before, negative after)Local Maximum at \(c\)
− → + (negative before, positive after)Local Minimum at \(c\)
+ → + or − → − (no sign change)Neither — it's an inflection point

The extremum value is \(f(c)\), not just "\(c\)". Always give the point \((c,\, f(c))\) in your final answer.

Second Derivative Test (SDT)

At a critical number \(c\) where \(f'(c)=0\):

\(f''(c) > 0 \Rightarrow\) Local Minimum (curve is concave up, like a ∪)
\(f''(c) < 0 \Rightarrow\) Local Maximum (curve is concave down, like a ∩)
\(f''(c) = 0 \Rightarrow\) Test is inconclusive — use FDT instead

Which test to use?

SituationBest Test
\(f''\) is easy to computeSDT (faster, no sign chart)
\(f''(c) = 0\) or \(f''\) is messyFDT (always works)
\(f'(c)\) undefined (e.g. cusp)FDT only — SDT requires \(f'(c)=0\)
Example 2 \(f(x) = x^3 - 3x + 2\) (continued)

Critical numbers: \(x = -1\) and \(x = 1\).

Using SDT: \(f''(x) = 6x\)

  • \(f''(-1) = -6 < 0\) → Local Maximum at \(x=-1\). Value: \(f(-1) = -1+3+2 = 4\). Point: \((-1,\,4)\).
  • \(f''(1) = 6 > 0\) → Local Minimum at \(x=1\). Value: \(f(1) = 1-3+2 = 0\). Point: \((1,\,0)\).

Sanity check with FDT: sign of \(f'\) goes + → − at \(x=-1\) ✓ and − → + at \(x=1\) ✓

🚨
Trick Question Alert: The function \(f(x) = x^4\) has \(f'(0) = 0\) and \(f''(0) = 0\), so SDT fails. But from FDT, \(f'(x) = 4x^3\) changes from − to + at \(x=0\), confirming a local (and absolute) minimum. Never stop at a failed SDT — always follow up with FDT.
🚨
Another Trap: \(f(x) = x^3\) has \(f'(0)=0\) but the sign of \(f'\) does NOT change at \(x=0\) (it's + on both sides). So \(x=0\) is not an extremum — it's an inflection point. Students often call it a minimum by mistake.
Chapter 03

Absolute (Global) Extrema

The absolute maximum is the single highest value \(f\) achieves over its entire domain (or a specified interval). The absolute minimum is the lowest. They may or may not exist depending on the domain.

Closed Interval Method (Exam Favourite)

If \(f\) is continuous on a closed bounded interval \([a,b]\), then:

  1. Find all critical numbers of \(f\) inside \((a,b)\).
  2. Evaluate \(f\) at every critical number AND at both endpoints \(a\) and \(b\).
  3. The largest value is the absolute max; the smallest is the absolute min.

Theorem guarantee: By the Extreme Value Theorem, a continuous function on \([a,b]\) always has both an absolute max and min.

Example 3 \(f(x) = x^3 - 3x + 2\) on \([-2, \frac{3}{2}]\)

Critical numbers in \((-2, \frac{3}{2})\): \(x=-1\) and \(x=1\) (both qualify).

Point\(x=-2\)\(x=-1\)\(x=1\)\(x=\tfrac{3}{2}\)
Type EndpointCriticalCriticalEndpoint
\(f(\cdot)\) \(-8+6+2=0\) \(4\) \(0\) \(\tfrac{27}{8}-\tfrac{9}{2}+2=\tfrac{11}{8}\)

Absolute Maximum: \(4\) at \(x=-1\)

Absolute Minimum: \(0\) at \(x=1\)

On an Open Interval or All of ℝ

The Extreme Value Theorem no longer guarantees existence. Use this approach instead:

  1. Find all critical numbers. If there is only one critical number \(c\) in the domain, and it is a local max (min), then it is also the absolute max (min). This is the single-critical-point test.
  2. Check the behaviour at the domain's boundaries: \(\lim_{x\to \text{edge}} f(x)\). If the function goes to \(\pm\infty\) at boundaries, the extremum can't be there.
  3. Compare all local extrema values to determine the global winner.
💡
Single Critical Point Rule

\(f(x) = e^{-x^2}\) has only one critical point, \(x=0\). Since \(f''(0) < 0\) → local max. As \(x\to\pm\infty\), \(f\to 0\). So \(f(0)=1\) is the absolute max.

⚠️
No Abs. Extremum — Common Cases

\(f(x)=x\) on \(\mathbb{R}\) — no absolute max or min.
\(f(x)=\frac{1}{x}\) on \((0,1)\) — no min (approaches 1 from above, never reached; unbounded above near 0).

✅
Local ≠ Absolute

A local max is only absolute if no other point on the function is higher. Always compare all local extrema values and boundary behaviour before concluding.

Chapter 04

Concavity & Inflection Points

Concavity describes the "curvature direction" of the graph. It's crucial for drawing an accurate sketch.

The Second Derivative Sign Rule
\(f''(x) > 0\) on \((a,b)\;\Rightarrow\) Concave Up (∪ shape — curve lies above its tangent lines)
\(f''(x) < 0\) on \((a,b)\;\Rightarrow\) Concave Down (∩ shape — curve lies below its tangent lines)

Inflection Points

An inflection point is where the concavity changes. Candidates: where \(f''(x) = 0\) or \(f''(x)\) is undefined. But you must confirm the sign actually flips.

🚨
\(f''(c)=0\) does NOT guarantee an inflection point. Example: \(f(x)=x^4\), \(f''(0)=0\), but the concavity does not change (always concave up). Always confirm the sign change!
Example 4 \(f(x) = x^3 - 3x + 2\) (continued)

\(f''(x) = 6x\). Set \(f''(x)=0\): \(x=0\).

Interval\((-\infty, 0)\)\(x=0\)\((0,+\infty)\)
\(f''(x)\)− (e.g. \(f''(-1)=-6\))0+ (e.g. \(f''(1)=6\))
ConcavityConcave Down ∩—Concave Up ∪

Sign changes at \(x=0\): Inflection Point at \((0,\, f(0)) = (0,\, 2)\).

🎯
Concavity Confirms Extrema

If you know a critical point and the concavity: local min ↔ concave up, local max ↔ concave down. Visualise it: a ball resting in a bowl (∪ = min) or balanced on a hill (∩ = max).

🔄
Higher-Order Test

If \(f'(c)=f''(c)=0\), keep differentiating. The first non-zero derivative \(f^{(n)}(c)\): if \(n\) is odd → inflection point; if \(n\) is even → extremum (min if positive, max if negative).

Chapter 05

Asymptotes

Asymptotes are lines the graph approaches but (usually) never touches. There are three types. Know all three cold.

Vertical
One (or both) sides approach \(\pm\infty\)
Horizontal
Function approaches a finite limit as \(x\to\pm\infty\)
Oblique (Slant)
Function approaches a non-horizontal line as \(x\to\pm\infty\)

Vertical Asymptotes

Finding Vertical Asymptotes
  1. Find values where the function is undefined (for rational functions: denominator = 0; for logs: argument ≤ 0; for tan: etc.).
  2. At each such \(x=a\), compute \(\lim_{x\to a^+} f(x)\) and \(\lim_{x\to a^-} f(x)\).
  3. If either one-sided limit is \(\pm\infty\), then \(x=a\) is a vertical asymptote.
  4. If both one-sided limits are the same finite value, it's a removable discontinuity (hole), not an asymptote.
VA Example \(f(x) = \dfrac{x+1}{x^2-1}\)

\(x^2-1 = (x-1)(x+1)\). Undefined at \(x=1\) and \(x=-1\).

At \(x = -1\): Factor: \(f(x) = \dfrac{x+1}{(x-1)(x+1)} = \dfrac{1}{x-1}\) when \(x\ne -1\). Limit as \(x\to-1\) is \(\dfrac{1}{-2} = -\tfrac{1}{2}\). This is a hole (removable discontinuity), NOT an asymptote.

At \(x = 1\): Simplified: \(\lim_{x\to 1}\dfrac{1}{x-1}\). From the right: \(+\infty\). From the left: \(-\infty\). So \(x=1\) is a vertical asymptote.

🚨
Huge Trap: Always simplify/factor first. If the numerator shares a factor with the denominator, that zero produces a hole, not an asymptote. Examiners love this.

Horizontal Asymptotes

Finding Horizontal Asymptotes

Compute \(\displaystyle\lim_{x\to+\infty} f(x)\) and \(\displaystyle\lim_{x\to-\infty} f(x)\). If either limit equals a finite number \(L\), then \(y = L\) is a horizontal asymptote.

Speed Rules for Rational Functions \(\dfrac{p(x)}{q(x)}\)

Degree comparisonHorizontal Asymptote
\(\deg p < \deg q\)\(y = 0\)
\(\deg p = \deg q\)\(y = \dfrac{\text{leading coeff of }p}{\text{leading coeff of }q}\)
\(\deg p > \deg q\)No horizontal asymptote (check for oblique)
💡
Key fact: A function can cross a horizontal asymptote (unlike vertical ones). The asymptote only describes end behaviour. Always check if the curve crosses by solving \(f(x) = L\).

Oblique (Slant) Asymptotes

Finding Oblique Asymptotes

Only possible when \(\deg(\text{numerator}) = \deg(\text{denominator}) + 1\).

  1. Perform polynomial long division: \(\dfrac{p(x)}{q(x)} = (mx + b) + \dfrac{r(x)}{q(x)}\).
  2. As \(x\to\pm\infty\), the remainder term \(\dfrac{r(x)}{q(x)}\to 0\).
  3. The oblique asymptote is \(y = mx + b\).
OA Example \(f(x) = \dfrac{x^2+1}{x-1}\)

Long division: Divide \(x^2+1\) by \(x-1\).

x² + 1 ÷ (x − 1)
x² ÷ x = x → x · (x−1) = x² − x → remainder: x + 1
(x+1) ÷ (x−1): x ÷ x = 1 → 1·(x−1) = x−1 → remainder: 2
Result: \(\dfrac{x^2+1}{x-1} = x + 1 + \dfrac{2}{x-1}\)

As \(x\to\pm\infty\), \(\dfrac{2}{x-1}\to 0\). So the oblique asymptote is \(y = x+1\).

Also: \(x=1\) is a vertical asymptote (denominator is zero, numerator is non-zero).

📐
Oblique for Non-Rationals

For non-rational functions, compute \(m = \lim_{x\to\infty}\frac{f(x)}{x}\) and \(b = \lim_{x\to\infty}[f(x)-mx]\). If both limits exist and \(m\ne 0\), then \(y=mx+b\) is oblique.

🔢
Two Different HAs Possible

\(f(x) = \frac{x}{|x|+1}\) has \(\lim_{x\to+\infty}=1\) and \(\lim_{x\to-\infty}=-1\). So there are two horizontal asymptotes: \(y=1\) and \(y=-1\).

⚠️
L'Hôpital's Rule

When limits give \(\frac{\infty}{\infty}\) or \(\frac{0}{0}\), use L'Hôpital: \(\lim\frac{f}{g}=\lim\frac{f'}{g'}\). Useful for limits involving \(e^x\), \(\ln x\), or tricky polynomials at infinity.

Chapter 06

Complete Graphing Strategy

Combine every tool above into a systematic procedure. Work through the checklist every time — never skip steps.

  1. Domain — Where is \(f(x)\) defined? State it. Any excluded values become candidates for vertical asymptotes or holes.
  2. Intercepts — \(y\)-intercept: set \(x=0\). \(x\)-intercepts: solve \(f(x)=0\).
  3. Symmetry — Test even (\(f(-x)=f(x)\)) or odd (\(f(-x)=-f(x)\)). Saves work.
  4. Asymptotes — Vertical (from domain gaps), Horizontal (limits at \(\pm\infty\)), Oblique (long division if needed). Mark all as dashed lines on the sketch.
  5. First Derivative \(f'\) — Find critical numbers. Make a sign chart. Determine increasing/decreasing intervals. Classify local extrema.
  6. Second Derivative \(f''\) — Find inflection candidates. Make sign chart. Determine concavity intervals. Confirm inflection points.
  7. Key Points — Compute \(f\) at all: intercepts, critical numbers, inflection points. Plot these.
  8. Sketch — Draw asymptotes first (as guides). Plot key points. Connect with curves that respect the increasing/decreasing direction and concavity. Curves approach asymptotes correctly.
How Concavity Shapes the Curve Between Points

Between two plotted points, the connecting curve must be:

DirectionConcavityShape of the arc
IncreasingUp (∪)Curve bends upward — accelerating rise
IncreasingDown (∩)Curve bends downward — decelerating rise
DecreasingUp (∪)Curve bends upward — decelerating fall
DecreasingDown (∩)Curve bends downward — accelerating fall
Chapter 07

Full Worked Example

Full Analysis \(f(x) = \dfrac{x^2}{x^2 - 4}\)
Step 1 — Domain & Vertical Asymptotes

\(x^2 - 4 = 0 \Rightarrow x = \pm 2\). Domain: \(\mathbb{R}\setminus\{-2,\,2\}\).

At \(x=2\): numerator \(=4\ne 0\), so \(x=2\) is a vertical asymptote. Similarly \(x=-2\) is a vertical asymptote.

Directions: near \(x=2^+\), denominator is small positive, numerator \(\approx 4>0\), so \(f\to+\infty\). Near \(x=2^-\), denominator small negative, so \(f\to-\infty\). Similarly for \(x=-2\).

Step 2 — Intercepts & Symmetry

y-intercept: \(f(0) = 0/(-4) = 0\). Point: \((0,0)\).

x-intercepts: \(f(x)=0 \Rightarrow x^2=0 \Rightarrow x=0\). So only \((0,0)\).

Symmetry: \(f(-x) = \dfrac{(-x)^2}{(-x)^2-4} = \dfrac{x^2}{x^2-4} = f(x)\). Even function — graph is symmetric about the \(y\)-axis. We only need to fully analyse \(x\ge0\) and mirror it.

Step 3 — Horizontal Asymptote

\(\displaystyle\lim_{x\to\pm\infty}\frac{x^2}{x^2-4} = \lim_{x\to\pm\infty}\frac{1}{1-4/x^2} = \frac{1}{1} = 1\)

\(y=1\) is the horizontal asymptote (both as \(x\to+\infty\) and \(x\to-\infty\)).

Does the curve cross the HA? Solve \(f(x)=1\): \(\dfrac{x^2}{x^2-4}=1 \Rightarrow x^2=x^2-4 \Rightarrow 0=-4\). No solution. The graph never crosses \(y=1\).

Step 4 — Monotonicity (f' Sign Chart)

Using the quotient rule:

\(f'(x) = \dfrac{2x(x^2-4) - x^2 \cdot 2x}{(x^2-4)^2} = \dfrac{2x^3-8x-2x^3}{(x^2-4)^2} = \dfrac{-8x}{(x^2-4)^2}\)

Critical numbers: \(f'(x)=0\) when \(-8x=0\Rightarrow x=0\). Undefined at \(x=\pm2\) (not in domain).

Interval\((-\infty,-2)\)\((-2,0)\)\(x=0\)\((0,2)\)\((2,+\infty)\)

Sign changes from + to − at \(x=0\): Local Maximum at \(x=0\). \(f(0)=0\).

No sign change at \(x=\pm2\) — they're not in the domain.

Increasing: \((-\infty,-2)\) and \((-2,0]\).  Decreasing: \([0,2)\) and \((2,+\infty)\).

Step 5 — Concavity (f'' Sign Chart)

Differentiate \(f'(x) = -8x(x^2-4)^{-2}\):

\(f''(x) = \dfrac{-8(x^2-4)^2 - (-8x)\cdot 2(x^2-4)\cdot 2x}{(x^2-4)^4}\) \(= \dfrac{-8(x^2-4) + 32x^2}{(x^2-4)^3} = \dfrac{24x^2+32}{(x^2-4)^3}\)

Numerator: \(24x^2+32 > 0\) always. Sign of \(f''\) determined by \((x^2-4)^3\):

Concavity changes at \(x=\pm2\), but those are not in the domain — no inflection points. No other \(f''=0\) candidates.

Step 6 — Summary & Sketch Description

Asymptotes: \(x=-2\), \(x=2\) (vertical); \(y=1\) (horizontal)

Key point: \((0,0)\) is a local max AND the only intercept.

The sketch: Draw dashed lines at \(x=\pm2\) and \(y=1\). On \((-\infty,-2)\): function increases from \(y=1^+\) (approaching HA from above since \(f\to1\) and concave up), going up to \(+\infty\) as \(x\to-2^-\). On \((-2,2)\): starts at \(-\infty\) (as \(x\to-2^+\)), rises through \((0,0)\) (local max), then falls to \(-\infty\) as \(x\to2^-\). Concave down throughout. On \((2,+\infty)\): starts at \(+\infty\), decreases toward \(y=1^+\), concave up.

Trick Example \(f(x) = x^{2/3}(x-5)\)

This function has a cusp — a place where \(f'\) is undefined but \(f\) is still defined. Examiners love these.

Full Analysis

Domain: All of \(\mathbb{R}\) (cube root is defined everywhere).

Intercepts: \(f(0)=0\); \(f(5)=0\). Points: \((0,0)\) and \((5,0)\).

Derivative: \(f'(x) = \frac{2}{3}x^{-1/3}(x-5) + x^{2/3} = \frac{2(x-5)}{3x^{1/3}} + x^{2/3}\)

Combine over common denominator \(3x^{1/3}\):

\(f'(x) = \dfrac{2(x-5) + 3x}{3x^{1/3}} = \dfrac{5x-10}{3x^{1/3}} = \dfrac{5(x-2)}{3x^{1/3}}\)

Critical numbers: \(f'(x)=0\) when \(x=2\); \(f'(x)\) undefined when \(x=0\) (but \(f(0)\) exists → it's a critical number too).

Interval\((-\infty,0)\)\(x=0\)\((0,2)\)\(x=2\)\((2,+\infty)\)
\(5(x-2)\)−−−0+
\(3x^{1/3}\)−0+++
\(f'(x)\)+⊘−0+
\(f\)↗Cusp↘—↗

At \(x=0\): sign changes + → −, so Local Maximum. \(f(0)=0\). And since \(f'\to\pm\infty\) here, it's a cusp (vertical tangent).

At \(x=2\): sign changes − → +, so Local Minimum. \(f(2)=4^{1/3}(-3)\approx -4.76\).
